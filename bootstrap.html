Bootstrap Icons

Icons speziell für das Bootstrap-Framework – aber überall verwendbar.

Einbindung in HTML

Bootstrap Icons lassen sich einfach über ein CDN einbinden:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.5/font/bootstrap-icons.css">

Beispiel für ein Icon

<i class="bi bi-alarm"></i>
