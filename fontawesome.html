Font Awesome Icons

So verwendest du Font Awesome in HTML.

Was ist Font Awesome?

Font Awesome ist eine der beliebtesten Icon-Bibliotheken für Webentwickler. Sie stellt hunderte von Icons zur Verfügung, die einfach per <i>-Tag eingebunden werden können.

Einbindung in HTML

Du kannst Font Awesome über ein CDN wie folgt einbinden:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

Beispiel für ein Icon

<i class="fas fa-camera"></i>
